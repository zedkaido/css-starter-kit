css-starter-kit (HTML5 Elements)

css-starter-kit is a suckless and sensible CSS starter kit to fix UI inconsistencies across browsers and improve accessibility.

Following are a series of HTML5 elements for testing purposes:

  • Text

    • Blockquotes
